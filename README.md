# Movie Recommendation System 🎬

A **Movie Recommendation System** built with **Machine Learning** that suggests movies based on user preferences.

## Features
- Suggests movies based on similarity with the selected movie.
- Fetches and displays movie posters dynamically.
- Easy-to-use interface with Streamlit.
